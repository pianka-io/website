Unity + Meta Quest on Apple Silicon

Dec 8, 2024 | 2 minutes read
Share this: Twitter Facebook Pinterest

Tags: unity, meta, quest, apple

If you’re looking to develop Unity projects for Meta Quest devices using macOS with an M-series chip, it recently got a little easier. This is a step-by-step guide to setting up a fresh environment with Unity, the Meta XR Simulator, your Meta Quest device, and an Xbox Wireless Controller on Apple Silicon. Before we begin, it’s important to note that Meta’s Unity Link tool is still not supported on macOS.

We’ll first install the Meta XR Simulator and its dependencies, then configure it in a new Unity 6 project, and map an Xbox controller to its input. Then we’ll hook Unity up to our Meta Quest device as a build and run target, and map its controls to the simulator as well.

Install Homebrew if you don’t already have it.

Meta XR Simulator

The simulator relies on the Vulkan SDK. Download it from here. Unzip it and run the installer app InstallVulkan.app. The options you select during installation are important!

Press Next on the welcome screen of the installer. Press Next again to use the default installation location (or selection a custom one if you’d like). On the components screen, make sure that System Global Installation is selected:

Press Next and continue the rest of the installation as normal.

Once that installation is complete, install the simulator itself in terminal:

brew tap Oculus-VR/tap
brew install meta-xr-simulator

Now we need to run the post installation script. First find the version of the simulator you just installed:

ls -la /opt/homebrew/Cellar/meta-xr-simulator/

Then use it by replacing __VERSION__ when running the script:

sudo /opt/homebrew/Cellar/meta-xr-simulator/__VERSION__/post_installation_macos.sh

At this point, the simulator should be fully installed and configured.

Unity

Grab the Unity Hub if you don’t already have it:

brew update
brew install --cask unity-hub

Open Unity Hub and go to Installs and then Install Editor. Select Install next to the latest version above 6000.0.30f1.
